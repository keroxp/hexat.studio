黒羽のアリカ

概要


『黒羽のアリカ』（くろばねのありか）は、@keroxpが個人で企画・開発しているスマートフォンゲームです。

タップとフリック操作で障害物を避けながら、翼ある少女アリカたちをできるだけ高く飛ばしてください。

ゲームには翼ある個性豊かな女の子のキャラクターがたくさん登場します！

2017年6月20日、AppStoreにて配信開始いたしました！

トレイラー

スクリーンショット

ストーリー

翼の生えた人々が暮らす世界、ワルディア。

クモナシの街のはずれにあるフギンの谷にひとりで暮らすアリカは、鋭い目つきととても珍しい真っ黒な羽毛がトレードマークの女の子。

そんな感じのアリカだけど、彼女の周りには個性豊かな友達がたくさん！

大好物のリンゴを買ってアップルパイを作るために、アリカは今日も星集めに飛び立つのでした。

キャラクター紹介

アリカ / ARIKA

フギンの谷に住む、黒いつばさの女の子。
目つきが鋭いのは生まれつきで、怒っているわけではない。
無口で物静かだけど、本当は心優しい性格。
手先が器用で、工作や裁縫、料理が得意。好物はリンゴとアップルパイ。

マリ / MARI

海辺に住む、カモメのつばさの女の子。
真面目な性格で、パトロールのボランティアの合間に、よくアリカの家に遊びに行っては世話を焼いている。
好きなことは、小さな妹と一緒に砂浜を散歩すること。

クルル / KURURU

高い山に住む、ツバメのつばさの女の子。
自由奔放な性格で、いつも空を飛び回っては楽しそうなことを探している。
いつもニコニコしているが、ニコニコしているうちにその理由を忘れてしまうらしい。
好きなことは散歩ときれいな石集め。

スズネ / SUZUNE

クモナシの街に住む、スズメのつばさの女の子。
やんちゃな性格で、よくアリカにいたずらをしている。
アリカとは小さい頃からの幼馴染で腐れ縁。
好きなことはファッションとセルフィー。

シロハ / SHIROHA

湖の邸宅に住む、白鳥のつばさの女の子。
クールな性格で、退屈なことがきらい。
高貴な家のお嬢様で、わがままで世事には疎い。
メイドさんがいないと着替えもできない。
趣味はフェンシング。
